#Overview

##What is Burstly?

Burstly offers a suite of tools to help mobile developers better monetize their apps. The tool set allows developers to run any type of content or messaging to its users, track in-app purchases from buttons or banners, cross-promote other apps, and to work with an extensive set of 3rd party ad partners to improve revenue.
